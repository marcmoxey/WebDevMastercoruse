Search Engine Optimization (SEO)

Rank-improvement Tips

  • Have relevant content on your page
  • Use keywords in your content
  • Mobile-friendly - avoid popups
  • Content accuracy
  • How fast your page loads
  • Page engagement - how long do users stay
  • Page accessibility
  • Custom 404 Page
  • Proper Meta Tags
  • Use HTTPS
  • Use proper headings - one h1, multiple h2-h6

Page status Code

  1. 1xx - still working
  2. 2xx - All good
  3. 3xx - Redirects
  4. 4xx - You made a mistake
  5. 5xx - Server had an issue
